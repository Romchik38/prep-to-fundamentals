Обобщение Pool
  - simple - пул, внутри которого захордкожено создание элементов.
  - drain - решение проблемы, когда в пуле заканчиваются элементы
  - poolify - вынесение создания массива за пределы функции pool
  - improved - в пул при создании вносится минимальное значение и максимальное
  - async - для осинхронной передачи элемента в callbacki
  - symbol - примешиваем к элемнтам symbol, чтобы принимать только "свои".

Drain
  - в обычном пуле, когда заканчиваются элементы, то он выдает undefind
  - в drain если нет элементов, то создается новый и уже он отдается.

Poolify
  - создается фабрика, которая будет возвращать новый элемент.
  - эта функция передается в функцию poolify, где создается нужный массив элементов и возвращается pool

Improved
  - задаются значения min и max
  - если приблизилось к min, то создаются новые
  - если превышем maх, то лишние удаляются.

Async
  - pool(obj) - передача в пул объекта obj
  - pool(cb) - для возврата из пула используется callback
  - если в пуле нет элементов, то callback будет откладываться и вызовется тогда, когда эл. появится
