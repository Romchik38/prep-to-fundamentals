mixin:
  - примешивание нового свойства вручную
  - примешивание через функцию

extend:
  extend - копирование свойст через свою функцию
  Object.Assign - через встроенную

behavior:
  lazy - обернуть метод объекта так, чтобы он вызывался отложенно

class:
  class extends - расширить существующий класс новым, чтобы использовать свойства и методы стараго класса

events:
  можно промешать к объекту свойства event emiter.

build:
  примешивание свойств массиву, через которые по индексу можно обращаться к элементам.
