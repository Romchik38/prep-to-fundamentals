Модель - набор типизированых 

Алгоритм - логическое пошаговое исполнение программы. 

Программа - набор программных компонентов, которые объеденены в одно целое для выполнения конкретной задачи. 

Синтаксис - правила оформления программного кода. 

Значение - данные, которые размещаются в памяти, имеют тип и в виде литерала присваются в идентификатор
Идентификатор - это имя, которое содержит ссылку на определенное место в памяти, где хранится значение
Переменная - идентификатор, значение котором можно поменять
Константа - идентификатор, значение которого нельзя поменять 
Литерал - представление значения в коде программы. 
Присвоение - связывание идентификатора и значения в виде литерала. 

Оператор - наименьшая единица программного кода, которая исполняется компилятором. 
Выражение - это синтаксическая конструкция, котороя состоит из идентификатора, оператора, значения
